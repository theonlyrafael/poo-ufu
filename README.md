# Programação Orientada a Objetos | POO

> Atividades e trabalhos desenvolvidos durante o terceiro e o quarto períodos do curso de Sistemas de Informação na Universidade Federal de Uberlândia (UFU).

> [!IMPORTANT]
> Este repositório reúne diversos projetos Java independentes. Após clonar o repositório, abra no VS Code a pasta específica da atividade que deseja executar, em vez de tratar `POO1`, `POO2` ou o repositório inteiro como um único projeto Java. Caso contrário, o editor pode não reconhecer corretamente a pasta de código-fonte e apresentar avisos relacionados aos pacotes das classes.

## Objetivos

Guardar as atividades e os trabalhos desenvolvidos nas duas disciplinas, preservando meu histórico de aprendizado em programação orientada a objetos e a evolução dos conceitos estudados ao longo do curso.

## Estrutura do Repositório

O repositório está dividido de acordo com cada disciplina:

* **[`POO1`](./POO1)**: reúne as primeiras atividades de programação orientada a objetos em Java, incluindo exercícios de classes, objetos, encapsulamento, herança, polimorfismo, exceções e coleções. Também contém um dos projetos mais interessantes que já contribui, sendo ele um jogo <kbd>Quiz</kbd>.

* **[`POO2`](./POO2)**: contém a continuação dos estudos, com práticas sobre estruturas de dados, generics, threads, testes automatizados, princípios SOLID e trabalhos avaliativos.
